# svm_diabetes_prediction
# 🧠 Support Vector Machine (SVM) from Scratch

## 📌 Overview

This project implements a **Support Vector Machine (SVM)** classifier from scratch using Python, without relying on machine learning libraries like scikit-learn.
It demonstrates a strong understanding of core ML concepts such as **hinge loss, gradient descent, and hyperplane optimization**.

---

## 🚀 Key Features

* Built SVM classifier from scratch using **NumPy**
* Implemented **hinge loss optimization**
* Used **gradient descent** for training
* Supports **binary classification**
* Fully modular and readable implementation

---

## 🛠️ Tech Stack

* Python
* NumPy
* Matplotlib (for visualization)

---

## ⚙️ How It Works

The model:

1. Initializes weights and bias
2. Uses **hinge loss function**:
   L = max(0, 1 - y(w·x + b))
3. Updates parameters using gradient descent
4. Finds the optimal separating hyperplane
